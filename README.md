# Squarespace Homepage Redesign

This repository contains the custom CSS used to redesign the homepage of a client’s Squarespace website.

## Overview

The client wanted improvements to the visual structure and presentation of their homepage while continuing to use the existing Squarespace theme. The solution was implemented entirely through custom CSS overrides.

The goal of the work was to improve:

- Layout consistency
- Visual hierarchy
- Text readability over images
- Section spacing and alignment
- Overall aesthetic polish

All styling changes were implemented in a single CSS file and injected through Squarespace's custom CSS system.

## Technologies

- Squarespace
- Custom CSS
